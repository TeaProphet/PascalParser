# PascalParser

Компилятор Pascal, реализованный на Python с помощью либы pyparsing.  
  
### Типы данных:
* сhar
* integer 
* boolean  
* Одномерный массив  
  
### Операции:
+, –, *, /, div, mod, =, <, >, <=, >=, not  

### Операторы:
* if then [else]
* for
* while
* do while
  
### Процедуры и функции:
Передача параметров простых типов (не массивов) только по значению

### Ввод/вывод:
* Write/WriteLn
* Read/ReadLn

### Системные функции:
* Inc 
* Dec
* Abs
